

# Social Media Chat UI

A modern, responsive **Social Media Chat User Interface** built using **pure HTML and CSS**.
This project mimics the layout and design of popular chat applications and is ideal for learning UI structuring, layout design, and CSS styling.

---

## Features

* Modern social-media–style chat interface
* Clean & beautified UI
* Well-structured HTML layout
* Pure HTML & CSS (No JavaScript)
* Sidebar chat list with search
* Chat messages (sent & received styles)
* Online status indicator

---

## Technologies Used

* **HTML5**
* **CSS3 (Flexbox)**
